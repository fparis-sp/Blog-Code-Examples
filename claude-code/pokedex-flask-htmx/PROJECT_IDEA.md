# Idea de Proyecto: Pokédex Web

## Lo que quiero construir

Una aplicación web de Pokédex donde pueda buscar y ver información de Pokémon.

## Features básicas que me gustaría tener

- Poder buscar un Pokémon por su nombre o número
- Ver una lista de Pokémon (al menos los primeros 151, la Gen 1 clásica)
- Al hacer click en un Pokémon, ver su información detallada:
  - Stats (HP, Attack, Defense, etc.)
  - Tipos (Fire, Water, Grass...)
  - Habilidades
  - Imagen/sprite
  - Altura y peso

- Que funcione bien en móvil y desktop

## Stack que quiero usar

- **Backend:** Flask (Python) - porque es simple y lo conozco
- **Frontend:** HTMX - quiero probar esto de hacer interactividad sin escribir JavaScript
- **Estilos:** Tailwind CSS - para hacer algo bonito rápido
- **API:** PokeAPI (https://pokeapi.co) - es gratuita y tiene toda la info de Pokémon

## Requisitos técnicos

- Quiero que tenga tests (pytest)
- Que sea fácil de deployar (Docker)
- Código limpio y bien estructurado

## Nivel de experiencia

Tengo conocimientos básicos de:
- Python y Flask
- HTML/CSS
- Git

Pero nunca he usado HTMX ni he integrado una API externa en un proyecto así.
